import {
  TreeViewBasic,
  TreeViewBorderless,
  TreeViewCondensed,
  TreeViewMultiSelect,
  TreeViewWithItemIcon,
  TreeViewWithActionBar,
  StyledIcon,
  TreeNode as Node,
} from "../../examples/components/ContentTree"
import findIndex from "lodash/findIndex"

<Overview>
  A content tree provides users with a way to navigate nested hierarchical
  information using a parent-child relationship model. It is located on the left
  side of the screen and cannot be combined with Side Navigation or Toolbar.
</Overview>

<CodeBlock
  title="Basic"
  code={TreeViewBasic}
  scope={{ TreeView, TreeViewItem }}
>
  Content Tree can be created using the components <code> TreeView </code> in
  the root and <code> TreeViewItem </code> for nodes.{" "}
  <code> TreeViewItem </code> takes <code> nodeId </code> which should be a
  unique number for each node and the <code> label </code> prop to display the
  node text.
</CodeBlock>

<CodeBlock
  title="Tree with Icons"
  code={TreeViewWithItemIcon}
  scope={{ TreeView, TreeViewItem }}
  noInline={true}
>
  Icons can be configured at the root level and node level and the state can be
  managed with callback functions <code> onNodeToggle </code> and
  <code> onNodeSelect</code>. An additional icon can be added through the prop <code>
    itemIcon
  </code>.
</CodeBlock>

<CodeBlock
  title="Condensed Tree"
  code={TreeViewCondensed}
  scope={{ TreeView, TreeViewItem }}
>
  Use class <code> list-group-condensed </code> to shrink the size of Tree and
  its items.
</CodeBlock>

<CodeBlock
  title="Borderless Tree"
  code={TreeViewBorderless}
  scope={{ TreeView, TreeViewItem }}
>
  Add a class of <code>.list-group-borderless</code> to your Tree view for a
  borderless variant.
</CodeBlock>

<CodeBlock
  title="Multi Selection Tree"
  code={TreeViewMultiSelect}
  scope={{ TreeView, TreeViewItem }}
>
  Multiple node selection can be enabled through prop <code> multiSelect </code>
  .
</CodeBlock>

<CodeBlock
  title="Tree with Actions"
  code={TreeViewWithActionBar}
  scope={{ TreeView, TreeViewItem, StyledIcon, Node, findIndex }}
  noInline={true}
>
  Using the props and callback functions from the <code> TreeView </code>{" "}
  component we perform more actions on each node.
</CodeBlock>

  query TreeViewQuery {
    TreeView: componentMetadata(displayName: { eq: "TreeView" }) {
      ...ComponentApi_metadata
    }
    TreeViewItem: componentMetadata(displayName: { eq: "TreeViewItem" }) {
      ...ComponentApi_metadata
    }
  }
`
