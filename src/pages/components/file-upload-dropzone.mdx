import { graphql } from "gatsby"
import {
  FileUploadDropZoneBasic,
  FileUploadDropZoneCustomValidation,
} from "../../examples/components/FileUploadDropZone"

<Overview>
  File Upload Dropzone component allows users to upload content from their
  computer or device into the application by dragging a file (or files) into a
  target zone.
</Overview>

<CodeBlock title="Example" code={FileUploadDropZoneBasic} noInline={true}>
  For a styled drag 'n' drop zone for uploading files use the{" "}
  <code>&lt;FileUploadDropZone&gt;</code> component with an <code>id</code>. The
  event <code> onFiles </code> fires when the files are added and it contains
  FileList object along with the current error value.
  <br />
  Uploaded files are validated using the input from props <code> multiple</code>,
  <code> maxFileCount </code> and <code> maxTotalFileSizeBytes </code> and an appropriate
  error message is displayed if any validation fails.
</CodeBlock>

<CodeBlock
  title="Custom validation"
  code={FileUploadDropZoneCustomValidation}
  noInline={true}
>
  By providing <code> validator </code> prop custom validation can be added for
  files as shown in the example. The function accepts FileList object and it
  should either return null or the error message. The default validation will be
  disabled when using <code> validator</code>.
</CodeBlock>

<LinkedHeading h="2" className="h1" id="file-upload-zone-accessibility">
  Accessibility
</LinkedHeading>
<p>
  User can interact with File Upload Dropzone also through keyboard using the
  tab key (or shift + tab when tabbing backwards). When the drop zone or browse
  link is in focus, pressing the enter/return or space keys will open the
  default file picker.
</p>
