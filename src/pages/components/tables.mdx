    <code> data</code>. Table headers are defined with the <code>columns</code> prop
    which is an array of objects.
    <br />
    <code> accessor </code> is a required key since it is the column identifier and{" "}
    <code> Header </code> key is used to set the column label, eg.
    {`[{ Header: 'First Name', accessor: 'firstName' }]`}. Use
    <code> sortBy </code> to enable sorting for any column.
  </p>
  <p>
    A column can be sorted by clicking on the header. Repeat this action to
    change the sorting order.
  </p>
</CodeBlock>
