<p>
  To enable advanced features like sorting, use the <code> DataTable </code>{" "}
  component along with the basic <code> Table </code> and additional components
  such as
  <code> TableHead, TableRow, TableBody and TableCell</code>.
  <br />
  <br />
  <p>
    The <code>DataTable</code> component requires two props:
    <code> columns </code> which takes an array of header data and
    <code> data </code> prop which takes an array of row data. The row data
    should match the header information passed to columns prop.
    <br />
    The objects passed in the array to the <code> column </code> prop has a
    required key <code> accessor </code> which is the column identifier and and
    the <code> Header </code> key is used to set the column label, eg.
    {`[{ Header: 'First Name', accessor: 'firstName' }]`}.
  </p>
</p>

<CodeBlock title="Table with Sorting" code={TableWithSorting} noInline={true}>

    Use the prop <code> hasSorting </code> to enable/disable the sorting on the

    <code> sortBy </code> can be used on the column object to control the sorting
    at the column level.
  </p>
  <p>
    A column can be sorted by clicking on the header. Repeat this action to

  </p>
