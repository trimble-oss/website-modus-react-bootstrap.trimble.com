<CodeBlock
  title="Table with Sorting"
  scope={{
    DataTable,
    TableContainer,
    TableHead,
    TableBody,
    TableCell,
    TableHeader,
    TableRow,
    Table,
  }}
  code={TableWithSorting}
  noInline={true}
>
  To enable advanced features like sorting, use the <code> DataTable </code>{" "}
  component along with the basic <code> Table </code> and additional components
  such as
  <code> TableHead, TableRow, TableBody and TableCell </code>.<p>
    The <code>DataTable</code> component needs two props:
    <code> columns </code> and
    <code> data</code>. The <code> data </code> prop is rendered as rows in the
    table and the <code>columns</code> prop which takes an array of objects
    displays table headings.
    <br />
    The objects passed in the array to the <code> column </code> prop has a
    required key <code> accessor </code> which is the column identifier and and
    the <code> Header </code> key is used to set the column label, eg.
    {`[{ Header: 'First Name', accessor: 'firstName' }]`}. Use
    <code> sortBy </code> prop in the object to enable sorting for a specific
    column.
  </p>
  <p>
    A column can be sorted by clicking on the header. Repeat this action to
    change the sorting order.
  </p>
</CodeBlock>
