<p>
  The <code> DataTable </code> component is a wrapper of <code> Table </code>{" "}
  component with more features that can be turned on/off anytime through the
  API.
  <br />
  <code>DataTable</code> requires two props:
  <code> columns </code> which takes an array of header data and
  <code> data </code> prop which takes an array of row data. The row data should
  match the header information passed to columns prop. The objects passed in the
  array to the <code> column </code> prop has a required key{" "}
  <code> accessor </code> which is the column identifier and and the{" "}
  <code> Header </code> key is used to set the column label, eg.
  {`[{ Header: 'First Name', accessor: 'firstName' }]`}.
</p>
